---

## Instalación local

```bash
# Clonar el repositorio
git clone https://github.com/metafuncion/cuandofokinpagan.git

# Entrar a la carpeta del proyecto
cd cuandofokinpagan/

# Abrir index.html en tu navegador
open index.html
```
